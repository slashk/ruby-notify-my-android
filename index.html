ruby-notify-my-android by slashk

send notifications to Notify My Android application

Send notifications to Android devices via the Notify My Android API. Details about the API are available at https://www.notifymyandroid.com/.

Dependencies

production requires "xml-simple" gem development requires "webmock" gem

Install

gem install ruby-notify-my-android
